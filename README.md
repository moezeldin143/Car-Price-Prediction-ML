# Car Price Prediction using Machine Learning

## 📌 Overview
This project focuses on predicting car prices using Machine Learning techniques. The workflow includes data exploration, preprocessing, feature engineering, model training, evaluation, and hyperparameter tuning to achieve the best prediction performance.

---

## 🎯 Objectives
- Analyze the car price dataset.
- Perform data cleaning and preprocessing.
- Build and compare multiple machine learning models.
- Optimize model performance using Cross Validation and Hyperparameter Tuning.
- Evaluate the final model using suitable performance metrics.

---

## 📂 Dataset
The project uses a car price dataset containing different vehicle features that influence the selling price.
